# 🏦 ATM Simulator (Python CLI Project)

A simple command-line based ATM simulation built using Python. This project demonstrates basic programming concepts such as loops, functions, input validation, and modular design.

---

## 🚀 Features

* 🔐 Secure transaction handling with input validation
* 💰 Check account balance
* ➕ Deposit money
* ➖ Withdraw money (with balance check)
* ❌ Prevents invalid and negative inputs
* 🔁 Continuous interaction using loop

---

## 🧠 Concepts Used

* Functions (modular programming)
* Conditional statements (`if-elif-else`)
* Loops (`while`)
* Exception handling (`try-except`)
* User input handling
* Basic financial logic
